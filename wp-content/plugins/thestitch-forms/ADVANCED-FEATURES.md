## 🚀 Installation & Setup

### 1. Activate Plugin
- Go to WordPress Admin
- Plugins menu
- Find "The Stitch Custom Forms"
- Click "Activate"

### 2. Access Customization
- WordPress Admin
- left sidebar: **Form Submissions**
- Click **"Customize"** tab

### 3. Setup Your Branding
- **Colors & Styling** - Pick your colors
- **Branding** - Adjust sizing and spacing
- **Labels & Text** - Customize all text
- **Email Settings** - Configure notifications
- **Live Preview** - See changes in real-time

### 4. Add Forms to Pages
Use shortcodes anywhere:

```
[bridal_consultation_form]
[dream_outfit_form]
```

**Or use WordPress block:**
- Add block → Search "The Stitch"
- Select form
- Insert into page

## 🎨 Customization Guide

### Quick Color Setup (2 minutes)

1. Open **Customization Panel** → **Colors & Styling**
2. Click color boxes to open color picker
3. Set these essential colors:
   - Primary Button Color (your brand)
   - Button Hover Color (darker shade)
   - Input Focus Color (matches primary)
4. Click **"Save All Settings"**
5. Done! Forms now match your brand

**Examples:**

```css

    font-family: 'Georgia', serif;

    font-weight: 700;
```

## 🔧 Technical Details

### Requirements
- WordPress 5.0+
- PHP 7.4+
- jQuery (included with WordPress)
- Modern browser (Chrome, Firefox, Safari, Edge)

### File Structure
```
thestitch-forms/
├── thestitch-forms.php          (Main plugin file)
├── assets/
│   ├── css/
│   │   ├── forms.css             (Form styling)
│   │   └── admin.css             (Admin panel styling)
│   └── js/
│       ├── forms.js              (Form functionality)
│       └── admin.js              (Admin interactivity)
└── Documentation files
```

### Hooks & Filters
- `wp_enqueue_scripts` - Load frontend assets
- `admin_enqueue_scripts` - Load admin assets  
- `admin_menu` - Add admin pages
- `wp_ajax_submit_bridal_form` - Handle bridal submissions
- `wp_ajax_submit_dream_outfit_form` - Handle outfit submissions
- `wp_init` - Register custom post type

## 🚨 Troubleshooting

### Forms not showing?
1. Activate plugin (Plugins menu)
2. Verify shortcode: `[bridal_consultation_form]`
3. Check page/post has shortcode
4. Clear browser cache (Ctrl+Shift+Delete)

### Customization not updating?
1. Click "Save All Settings" (don't just refresh)
2. Hard refresh page (Ctrl+Shift+R)
3. Clear WordPress cache plugins (if using cache)
4. Check form is not in a cached page builder

### Email not received?
1. Check recipient email in Email Settings
2. Check spam/junk folder
3. Contact hosting about email limits
4. Verify admin email is valid

### Colors not changing?
1. Make sure colors are valid hex codes
2. Check color picker saved properly
3. Hard refresh browser
4. Try different browser
5. Disable conflicting plugins
