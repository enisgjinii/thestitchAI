## ✅ HOW TO ADD FORMS TO PAGES

### Method 1: WordPress Block Editor (Gutenberg)
```
1. Edit Page
2. Click + button
3. Search "Custom HTML" block
4. Paste: [bridal_consultation_form]
5. Update/Publish
```

### Method 2: Elementor
```
1. Open Elementor editor
2. Search "HTML" widget
3. Drag to page
4. Paste shortcode in HTML field
5. Save & Publish
```

### Method 3: Divi
```
1. Edit page with Divi
2. Add TEXT/CODE module
3. Paste shortcode
4. Save & Publish
```

### Method 4: WPBakery/Visual Composer
```
1. Edit page
2. Add "Raw HTML" element
3. Paste shortcode
4. Save
```

### Method 5: Text/Code Editor
```
1. Switch to HTML/Text mode
2. Paste shortcode where needed
3. Save page
```
