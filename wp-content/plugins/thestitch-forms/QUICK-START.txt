❓ TROUBLESHOOTING
==================

PROBLEM: Form not showing on page
SOLUTION: Check shortcode spelling
- [bridal_consultation_form] ✓ (correct)
- [Bridal_consultation_form] ✗ (case sensitive)
- [bridal-consultation-form] ✗ (wrong format)

PROBLEM: "Not submitting" / "Error submitting"
SOLUTION: 
- Ensure plugin is Activated
- Check browser console (F12) for errors
- Verify site has SSL (https://)
- Check WordPress AJAX is enabled

PROBLEM: File upload not working
SOLUTION:
- Ensure /wp-content/uploads/ has write permissions
- Check file size (max 5MB)
- Use supported formats (JPG, PNG, GIF, WebP)
- Ask hosting provider to check upload limits

PROBLEM: No email notifications
SOLUTION:
- Check admin email in Settings → General
- Check spam folder
- Verify submission saved in Form Submissions
- Ask hosting to enable mail functionality

PROBLEM: Forms look broken/ugly
SOLUTION:
- Clear browser cache (Ctrl+Shift+Del)
- Deactivate other form plugins temporarily
- Check for conflicting styles in theme
- Contact support if issue persists
