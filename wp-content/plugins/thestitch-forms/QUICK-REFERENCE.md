# 🚀 The Stitch Forms - Quick Start Card

## What You Have
✅ Full-featured WordPress forms plugin with admin customization panel

## 3-Step Activation
1. WordPress Admin → Plugins → Activate "The Stitch Custom Forms"
2. Verify activation (should see "Form Submissions" in sidebar)
3. Done! Plugin is active

## Add Forms to Your Website
Use these shortcodes:

```
[bridal_consultation_form]
[dream_outfit_form]
```

**Where to use:**
- In posts/pages
- In page builders (Elementor, Divi, etc.)
- In custom code

## Customize Your Forms
**WordPress Admin → Form Submissions → Customize**

### What You Can Change
| What | Where | Time |
|------|-------|------|
| Colors | Colors & Styling tab | 2 min |
| Form titles | Labels & Text tab | 2 min |
| Button text | Labels & Text tab | 2 min |
| Email settings | Email Settings tab | 3 min |
| Form sizing | Branding tab | 3 min |
| Advanced CSS | Branding tab (Custom CSS field) | 5+ min |

## Customization Panel Tabs

### 🎨 Tab 1: Colors & Styling
- Button color
- Input borders
- Success/error colors
- Text color

### 🏢 Tab 2: Branding
- Form width
- Border radius
- Padding/spacing
- Custom CSS

### 📝 Tab 3: Labels & Text
- Form titles
- Button labels
- Field placeholders

### 📧 Tab 4: Email Settings
- Recipient email
- Subject lines
- Auto-reply message

### 👁️ Tab 5: Live Preview
- See changes in real-time
- Test colors
- No refresh needed

## Color Setup (5 minutes)

1. Go to customization panel
2. Click "Colors & Styling" tab
3. Click color fields to open color picker
4. Select your brand colors
5. Click "Save All Settings"
6. Done!

## Common Tasks

### Change Submit Button Text
1. Labels & Text tab
2. Edit "Submit Button Text"
3. Save

### Change Form Title
1. Labels & Text tab
2. Edit "Form Title"
3. Save

### Make Forms Narrower
1. Branding tab
2. Set "Form Container Width" to 400-500px
3. Save

### Make Forms Wider
1. Branding tab
2. Set "Form Container Width" to 800-1000px
3. Save

### Round Corners
1. Branding tab
2. Set "Border Radius" to 15-20px
3. Save

### Sharp Corners
1. Branding tab
2. Set "Border Radius" to 0px
3. Save

## Email Notifications

### Getting Started
1. Email Settings tab
2. Verify recipient email
3. Edit subject lines if desired
4. Save

### Check It Works
1. Submit a test form
2. Check inbox for notification
3. Check spam folder if not found
4. Verify auto-reply message

## Troubleshooting

### Forms not showing?
- Check shortcodes are correct
- Verify plugin is activated
- Clear browser cache

### Changes not applying?
- Click "Save All Settings"
- Hard refresh browser (Ctrl+Shift+R)
- Check CSS variables loaded

### Email not working?
- Verify recipient email is valid
- Check spam folder
- Test form submission

### Can't find customization panel?
- WordPress Admin → Form Submissions
- Click "Customize" tab
- Should show 5 tabs
