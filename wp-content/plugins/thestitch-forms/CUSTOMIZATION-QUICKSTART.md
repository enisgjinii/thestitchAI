# ⚡ Form Customization Panel - Quick Start

## 3-Minute Setup

### Step 1: Access the Panel
WordPress Admin → **Form Submissions** → **Customize** tab

### Step 2: Choose Your Colors
**Colors & Styling Tab**
- Primary Button Color: Pick your brand color
- Button Hover Color: Slightly darker shade
- Input Focus Color: Same as primary button
- Click color boxes to open color picker

### Step 3: Customize Text
**Labels & Text Tab**
- Edit form titles and button labels
- Change placeholder text
- Match your brand voice

### Step 4: Check Preview
**Live Preview Tab**
- See real-time changes
- Test button styling
- Verify text looks good

### Step 5: Save & Done!
Click **"Save All Settings"** button

✓ Your forms now display with your branding!

---

## 5-Minute Deep Dive

### Colors to Change

**Must-Change:**
1. Primary Button Color → Your brand color
2. Button Hover Color → Darker version

**Nice-to-Change:**
3. Input Focus Color → Same as primary
4. Text Color → Match your site
5. Background Color → Blend with website

### Text to Change

**Recommended:**
- Form titles (e.g., "Book a Consultation")
- Button text (e.g., "Schedule Now")
- Email subject lines

**Optional:**
- Field placeholders
- Auto-reply messages

### Sizing Tips

**Form Width:**
- Mobile-focused: 400px
- Standard: 600px (default)
- Spacious: 800px

**Border Radius:**
- Sharp: 0px
- Modern: 8px (default)
- Rounded: 15px

## Common Changes

### 1️⃣ Match Brand Colors
1. Colors & Styling tab
2. Set Primary Button Color = your brand
3. Set Hover Color = darker shade
4. Save

### 2️⃣ Change Button Text
1. Labels & Text tab
2. Edit "Submit Button Text" fields
3. Save

### 3️⃣ Adjust Email Notifications
1. Email Settings tab
2. Update recipient email
3. Edit subject lines
4. Customize auto-reply message
5. Save

## Preview Tips

- **Live Preview updates automatically**
- **Don't refresh page while editing**
- **Test on mobile** to verify responsive
- **Hover over buttons** to see hover color

---

## Save & Reload

After saving:
1. Hard refresh browser (Ctrl+Shift+R)
2. Check website to see changes
3. Forms display with new styling!

---

## Revert to Defaults

To go back to original styling:
1. Customization Panel
2. Reload page (Ctrl+F5)
3. Manually change colors back to defaults:
   - Primary: #8b7355
   - Hover: #6d5a47
   - Text: #333333
