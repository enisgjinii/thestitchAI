🚀 HOW TO USE

STEP 1: Activate
─────────────────
1. Log into WordPress Dashboard
2. Go to: Plugins → Installed Plugins
3. Find: "The Stitch Custom Forms"
4. Click: "Activate"
5. Done! Plugin is active

What Happens:
- "Form Submissions" appears in admin menu
- "How to Use" page appears in admin
- All forms ready to use

STEP 2: Add to Pages
────────────────────
Copy one of these shortcodes:

For Bridal Consultation:
    [bridal_consultation_form]

For Dream Outfit Submission:
    [dream_outfit_form]

Then paste into any page using:
- WordPress Block Editor (Gutenberg)
- Elementor
- Divi
- WPBakery
- Text/Code editor
- Any WordPress page editor

STEP 3: Test
────────────
1. Submit test form
2. Check admin email
3. View in "Form Submissions"
4. Test mobile view
5. Verify file uploads

STEP 4: Go Live
───────────────
Share forms with customers via:
- Homepage
- Bridal Services page
- Custom Design page
- Contact page
- Services page
- Any relevant page

💾 DATA STORAGE

Where Data is Saved:
1. WordPress Database
   - Custom Post Type: "form_submission"
   - Can be filtered and sorted
   - Viewable in WordPress admin
   - Never deleted by updates

2. Email Notifications
   - Sent to site admin email
   - Contains form summary
   - File information
   - Timestamp

3. File Storage
   - Location: /wp-content/uploads/thestitch-forms/
   - Organized by submission
   - Maintains original filenames
   - Secure storage

Accessing Submissions:
1. WordPress Admin Dashboard
2. Click "Form Submissions" in menu
3. View all submissions
4. Click any to see full details
5. See all fields, files, metadata
