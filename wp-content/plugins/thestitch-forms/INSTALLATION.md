# Quick Start Guide - The Stitch Custom Forms

## 1️⃣ Activate the Plugin

1. Log in to your **WordPress Dashboard**
2. Go to **Plugins** → **Installed Plugins**
3. Find **The Stitch Custom Forms**
4. Click **Activate**

You're done! The plugin is now active.

## 2️⃣ Add Forms to Your Pages

### Method A: Using Page Editor

1. Go to **Pages** and edit the page where you want the form
2. Click to add your content
3. Paste one of these shortcodes:
   - **Bridal Consultation:** `[bridal_consultation_form]`
   - **Dream Outfit Submission:** `[dream_outfit_form]`
4. Click **Update/Publish**

### Method B: Using Elementor (if installed)

1. Edit page in Elementor
2. Search for **HTML** element
3. Paste shortcode in the HTML block
4. Save & Publish

### Method C: Using Gutenberg (WordPress Block Editor)

1. Edit page
2. Add a **Custom HTML** block
3. Paste shortcode:
   ```
   [bridal_consultation_form]
   ```
4. Publish

## 4️⃣ View Submissions

1. Go to **WordPress Admin**
2. Look for **Form Submissions** in the sidebar menu
3. Click to see all customer submissions
4. Click on any submission to view full details (files, measurements, etc.)

## 5️⃣ Get Email Notifications

Each time a customer submits a form:
- ✉️ Admin email is sent automatically
- 📊 Submission is saved to WordPress database
- 👁️ You can review in **Form Submissions** menu
